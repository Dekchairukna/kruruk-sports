# FISHBONE BRACKET CENTER ALIGN

ปรับก้างปลา/สายเข้ารอบให้รอบถัดไปอยู่กึ่งกลางระหว่างคู่หรือสายที่ส่งผู้ชนะเข้ามา ตามรูปแบบ bracket ใน Excel/เอกสารการแข่งขันจริง

## แก้ไข
- เพิ่ม class `level-0`, `level-1`, `level-2` ... ให้แต่ละรอบใน `templates/events/program.html`
- ใช้ namespace ใน Jinja เพื่อนับเฉพาะรอบที่แสดงจริง ไม่รวมรอบชิงอันดับ 3 ที่ถูกซ่อน
- Champion ใช้ level เดียวกับรอบสุดท้าย เพื่อให้อยู่ตรงกลางแนวเดียวกับคู่ชิง
- ปรับ CSS ใน `static/css/app.css` ให้รอบถัดไปมี offset แนวตั้งตามระดับรอบ
- รักษาขนาดกล่องเดิม แต่จัดตำแหน่งใหม่ให้เหมือน bracket ทางการมากขึ้น

## ผลลัพธ์
- รอบ 8/4/2 จะไม่วางเป็นแถวตรง ๆ อีก
- คู่รอบรองจะอยู่กลางระหว่างสองคู่รอบก่อนหน้า
- คู่ชิงจะอยู่กลางระหว่างสองคู่รอบรอง
- ถ้วยชนะเลิศอยู่แนวเดียวกับรอบชิง

## ตรวจสอบ
- `python -m py_compile app.py` ผ่าน
- Jinja parse `templates/events/program.html` ผ่าน
- Local default port 5050
