คู่มือแบบเร็ว

KRURUK SPORTS ใช้งานอย่างไร

ออกแบบให้ผู้จัดการแข่งขันทำงานเป็นลำดับ: สร้างงาน → เพิ่มทีม → เลือกกีฬา → จัดโปรแกรม → บันทึกผล → พิมพ์เอกสาร

1. Super Admin

ดูภาพรวมทั้งระบบ จัดการองค์กร ผู้ใช้ แพ็กเกจ และสิทธิ์การใช้งาน

2. Admin องค์กร

สร้างและจัดการเฉพาะงานของตนเอง เพิ่มทีม/สี สร้างรายการกีฬา และดูรายงาน

3. ครูประจำสี / ทีม

เข้าด้วยรหัสทีมเพื่อกรอกข้อมูล ดูผล และพิมพ์เกียรติบัตรเฉพาะทีมตัวเอง

เส้นทางทำงานที่แนะนำ

  1. สร้างอีเว้นท์ เลือกบริบท เช่น โรงเรียน กลุ่มโรงเรียน อบต. เทศบาล หรือองค์กร
  2. เพิ่มทีม/สี ระบบสร้างรหัสให้ทีมเข้ากรอกข้อมูลเองได้
  3. สร้างการแข่งขัน เลือกกีฬา เลือกทีมที่ส่งจริง และกำหนดระบบแข่ง
  4. โปรแกรม/จัดแข่ง ดูคู่แข่งขัน บันทึกผล ดูตารางคะแนน และจัดรอบต่อ
  5. สรุปผล ออกเกียรติบัตร รายงาน โปรแกรมแข่ง และสูจิบัตร
