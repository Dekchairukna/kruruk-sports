# EVENT_ADD_LIMIT_FIX_NOTES

แก้ปัญหาเพิ่มอีเว้นท์ไม่ได้

## สาเหตุที่พบ
- ฟังก์ชัน `check_event_limit()` ตรวจลิมิตแพ็กเกจ Free ซึ่งตั้งไว้ให้สร้างงานแข่งขันได้สูงสุด 1 งาน
- ทำให้ Super Admin ที่ต้องสร้างงานหลายงาน/หลายองค์กรถูกบล็อกเหมือนลูกค้าทั่วไป
- เพิ่มการกัน error กรณีฐานข้อมูลยังไม่มีแพ็กเกจเริ่มต้น เพื่อไม่ให้เกิด 500 จาก `plan.name`

## สิ่งที่แก้
- ให้ `superadmin` ผ่านลิมิตอัตโนมัติใน:
  - `check_event_limit()`
  - `check_team_limit()`
  - `check_athlete_limit()`
- ถ้าไม่พบแพ็กเกจ จะคืนข้อความเตือนแทนการทำให้ระบบพัง
- แก้ทั้ง `app.py` หลัก และ `kruruk_sports_phase0/app.py`

## ผลลัพธ์
- Super Admin เพิ่มอีเว้นท์ใหม่ได้แม้องค์กรนั้นมีอีเว้นท์เดิมแล้ว
- Super Admin เพิ่มทีม/นักกีฬาได้ไม่ติดลิมิต Free
- ผู้ใช้ทั่วไปยังถูกจำกัดตามแพ็กเกจเหมือนเดิม
