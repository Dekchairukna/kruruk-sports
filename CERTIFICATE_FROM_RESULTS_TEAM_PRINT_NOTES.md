# Certificate from Results + Team Print Flow

ปรับระบบเกียรติบัตรให้ยึดผลการแข่งขันจริงเป็นแหล่งข้อมูลหลัก

## Flow ใหม่
1. แอดมินบันทึกผลการแข่งขันและสรุปอันดับให้เรียบร้อย
2. เข้าเมนู Certificate ของอีเว้นท์
3. กด "ดึงผลและสร้างเกียรติบัตร"
4. ระบบสร้างรายชื่อเกียรติบัตรจากผลอันดับ 1-3 อัตโนมัติ
5. ครูประจำสีเข้าหน้า team-entry ด้วยรหัสทีม/สี
6. กด "พิมพ์เกียรติบัตร" เพื่อพิมพ์เฉพาะของทีม/สีตัวเอง

## แหล่งผลที่ดึง
- RankingCompetition / RankingResult
- ContestCompetition / ContestResult
- RoundRobinCompetition เฉพาะกรณีแข่งครบทุกคู่
- KnockoutCompetition เฉพาะรอบชิงที่มีผลแล้ว และรองรับนัดชิงอันดับ 3 ถ้ามี

## การจัดการข้อมูล
- ถ้ายังไม่มี Template ระบบจะสร้าง Template เริ่มต้นให้
- การซิงก์ผลจะลบ recipient_type=winner ของ template นั้น แล้วสร้างใหม่จากผลล่าสุด
- รายชื่อไม่ต้องกรอกเองสำหรับผู้ได้รับรางวัล
- รายชื่อเองยังใช้ได้สำหรับกรรมการ ผู้สนับสนุน หรือหน้าที่พิเศษ

## หมายเหตุ
- ปิดสิทธิ์กรอกข้อมูลทีมแล้ว ครูประจำสียังสามารถใช้รหัสเดิมเข้าหน้าพิมพ์เกียรติบัตรได้
- หน้า profile ทีมเพิ่มปุ่ม "พิมพ์เกียรติบัตร"
