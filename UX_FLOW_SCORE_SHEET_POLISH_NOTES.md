# UX Flow + Score Sheet Polish

ปรับรอบนี้ตามฟีดแบ็ก:

- แก้ Sidebar ไม่ให้ footer ทับเมนูด้านล่าง และปรับกลุ่มเมนูให้ดูเป็นระบบทดลองใช้มากขึ้น
- เปลี่ยนชื่อเมนูระบบให้เป็นภาษาไทยมากขึ้น: การชำระเงิน, บริบทงานทั้งหมด, แพ็กเกจ, ตั้งค่า
- หน้า Event เปลี่ยนส่วน “การแข่งขันในอีเว้นท์นี้” เป็นกล่องเส้นทางเดียว ไม่แยกการ์ดให้สับสน
- ในกล่องการแข่งขัน แสดงขั้นตอนต่อเนื่อง: ตั้งค่ารายการ → ทีม/สาย → บันทึกผล → จัดรอบต่อ → สรุปผล
- ปรับหน้าใบบันทึกคะแนนรายเซตให้เป็น Score Sheet มากขึ้น ไม่ขึ้นเลขตัวอย่างในช่องคะแนน
- เพิ่มข้อความกำกับ “ช่องว่างถือเป็น 0 เมื่ออีกฝั่งมีคะแนน”
- ใช้ลิมิตตามกีฬาเดิมใน backend/frontend:
  - เซปักตะกร้อ: 15 คะแนน, สูงสุด 17
  - วอลเลย์บอล: ต้องชนะห่าง 2 แต้ม, ไม่มีเพดานสูงสุดตามกติกาสากล
  - แบดมินตัน: สูงสุด 30
  - เทเบิลเทนนิส: ต้องห่าง 2 แต้ม, ไม่มีเพดาน
- Local default port ยังคง 5050

อ้างอิงแนว UI/แบบบันทึกคะแนนจากหน้าตัวอย่างที่ผู้ใช้ส่งมา แต่ไม่ได้คัดลอกไฟล์หรือรูปแบบเฉพาะของเว็บต้นทาง
