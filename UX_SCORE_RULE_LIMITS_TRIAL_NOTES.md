# UX + Score Rule Limits + Trial Mode Update

ปรับตามข้อเสนอครูรัก 2026-06-19

## 1) Sidebar / Dashboard
- ซ่อนเมนู Organizations จากผู้ใช้ทั่วไป ไม่ให้ลูกค้าต้องสร้างองค์กรเอง
- ให้สร้างบริบทงานจากหน้า "สร้างงานแข่งขัน" แทน เช่น โรงเรียน / กลุ่มโรงเรียน / อบต./เทศบาล / องค์กร
- Dashboard ทำหน้าที่เป็นหน้ารวมงานหลัก แยกกลุ่มงานเป็น:
  - งานที่กำลังเปิดใช้งาน
  - งานร่าง / เตรียมจัด
  - งานที่ปิดแล้ว / เก็บถาวร
- เพิ่ม badge "ทดลองใช้"
- ปรับสีเป็น pastel blue/violet ดูน่าใช้ขึ้น

## 2) Port
- Local default เปลี่ยนเป็น 5050
- Production/Railway ยังใช้ PORT environment ตามเดิม

## 3) หน้าบันทึกคะแนนรายเซต
- คืน flow หน้าบันทึกคะแนนสำหรับกีฬารายเซต เช่น วอลเลย์บอล/ตะกร้อ/แบดมินตัน/เทเบิลเทนนิส
- ถ้ารายการเก่าเป็น score_only แต่ชื่อกีฬาเป็นวอลเลย์บอล/ตะกร้อ ระบบจะแสดงเป็น set_based ให้อัตโนมัติ
- เอา placeholder ตัวเลขตัวอย่างออกจากช่องคะแนน
- เพิ่มใบคะแนนแบบกีฬานั้น ๆ พร้อมคำอธิบายกติกาที่หน้า score sheet
- กดเพิ่มแต้มแล้วมี limit กันกดค้าง:
  - เซปักตะกร้อ: สูงสุด 17 แต้ม
  - แบดมินตัน: สูงสุด 30 แต้ม
  - วอลเลย์บอล: ไม่มีเพดานสูงสุดตามกติกา แต่ต้องห่าง 2 แต้ม และระบบกันเพิ่มหลังเซตจบแล้ว
  - เทเบิลเทนนิส: ไม่มีเพดานสูงสุด แต่ต้องห่าง 2 แต้ม และระบบกันเพิ่มหลังเซตจบแล้ว
- Backend ตรวจซ้ำก่อนบันทึก กันข้อมูลเกินกติกาแม้แก้จาก HTML

## 4) อ้างอิงกติกา
- Volleyball: FIVB Official Volleyball Rules 2025-2028
- Sepaktakraw: ISTAF Law of the Game 2024 / 15-point scoring system
- Badminton: BWF 21-point scoring rules with cap at 30
