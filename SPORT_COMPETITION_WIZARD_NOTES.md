# ปรับ Flow การสร้างรายการแข่งขันตามมุม User

## ปัญหาเดิม
ระบบเดิมออกแบบแบบโครงสร้างข้อมูล คือ หมวดกีฬา → ชนิดกีฬา → รายการย่อย/รุ่น → ค่อยไปสร้างระบบแข่ง ทำให้ผู้ใช้รู้สึกว่า “ระบบติดตัวกีฬา” และต้องตั้งค่าหลายชั้นก่อน ทั้งที่ตอนจัดงานจริงต้องการเพียง:

1. เลือกกีฬาที่จะจัดจริง เช่น วอลเลย์บอล
2. เลือกทีม/สีที่ส่งรายการนี้จริง เพราะบางสีอาจไม่ส่ง
3. ดูจำนวนทีม แล้วค่อยเลือกระบบแข่ง
4. ระบบแนะนำเซต/รอบ/กลุ่มตามชนิดกีฬาและจำนวนทีม

## สิ่งที่เพิ่ม
เพิ่มหน้าใหม่:

- `/events/<event_id>/competitions/quick-new`
- endpoint: `event_competition_wizard`
- template: `templates/sports/competition_wizard.html`

## Flow ใหม่
หน้าใหม่ทำงานแบบ Wizard 3 ขั้น:

### 1) เลือกกีฬา
มีปุ่มลัด เช่น วอลเลย์บอล, ฟุตซอล, ฟุตบอล, ตะกร้อ, เปตอง, ชักเย่อ, กรีฑา, ประกวดกองเชียร์

ผู้ใช้ไม่จำเป็นต้องไปสร้างชนิดกีฬาก่อน ถ้ากีฬายังไม่มีใน Event ระบบจะสร้าง Sport/SportDivision ให้เองตอนกดสร้างรายการ

### 2) เลือกทีมที่ส่งจริง
แสดง Team/สีทั้งหมดจาก Event นั้น แล้วให้ติ๊กเฉพาะทีมที่ส่งรายการนั้น
เช่น วอลเลย์บอล ป.6 หญิง อาจมีแค่ สีแดง สีฟ้า สีเขียว ส่ง แต่สีเหลืองไม่ส่ง ก็ไม่ต้องสร้างคู่ให้สีเหลือง

### 3) เลือกระบบแข่งและกติกา
ระบบแนะนำตามจำนวนทีม:

- 2 ทีม → Knockout ชิงทันที
- 3–5 ทีม → Round Robin 1 กลุ่ม
- 6–8 ทีม → Round Robin 2 กลุ่ม เข้ารอบกลุ่มละ 2
- 9–12 ทีม → Round Robin 3 กลุ่ม ที่ 1 แต่ละกลุ่ม + รองดีที่สุด 1
- 13 ทีมขึ้นไป → Round Robin 4 กลุ่ม เข้ารอบกลุ่มละ 2
- กรีฑา/วิ่ง → Ranking
- กิจกรรมประกวด → Score Judging

สำหรับวอลเลย์บอล ระบบตั้งค่าเริ่มต้นเป็น:

- Set Based
- 5 เซต
- 25 แต้ม/เซต
- ชนะ 3 เซต

## ไฟล์ที่แก้
- `app.py`
  - เพิ่ม route `event_competition_wizard`
  - สร้าง RoundRobinCompetition พร้อมกลุ่มและแมตช์จากทีมที่เลือก
  - สร้าง KnockoutCompetition พร้อมรอบแรกจากทีมที่เลือก
  - สร้าง Ranking/Contest จากหน้าเดียวกันได้
- `templates/sports/competition_wizard.html`
  - หน้า wizard ใหม่
- `templates/sports/setup.html`
  - เพิ่มปุ่ม “สร้างรายการแข่งจากทีมที่ส่งจริง”
  - เปลี่ยนปุ่มชุดกีฬาเริ่มต้นเป็น “คลังตัวอย่างกีฬา” เพื่อลดความรู้สึกว่าระบบบังคับติดกีฬา
- `templates/events/detail.html`
  - เพิ่มปุ่ม “สร้างรายการแข่ง” ในหน้างานแข่งขัน

## หมายเหตุ
ของเดิมยังอยู่ทั้งหมด ไม่ลบ route เดิม ไม่ลบระบบตั้งค่ากีฬาเดิม เพื่อไม่กระทบข้อมูลและงานที่ทำไว้แล้ว
