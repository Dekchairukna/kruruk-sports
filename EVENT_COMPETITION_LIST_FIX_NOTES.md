# Event Competition List Fix

ปรับหน้า Event ตาม feedback ล่าสุด:

## ปัญหาเดิม
- ตัดปุ่ม Round Robin / Ranking / Knockout ออกแล้ว แต่ไม่ได้เพิ่มส่วนแสดงรายการแข่งขันที่สร้างแล้ว
- หลังสร้างการแข่งขัน ผู้ใช้ไม่เห็นว่าจะกลับเข้าไปจัดการผล/ตาราง/รอบต่อไปตรงไหน
- ปุ่มบนหน้า Event ขนาดไม่เท่ากัน ดูเหมือนพัง

## สิ่งที่แก้
1. หน้า Event เหลือปุ่มหลัก “สร้างการแข่งขัน” เป็นทางหลัก
2. เพิ่ม section ใหม่ “การแข่งขันในอีเว้นท์นี้” ใต้การ์ดข้อมูลงาน
3. ดึงรายการที่สร้างแล้วทั้งหมดมาแสดงรวมในหน้าเดียว:
   - Round Robin / แบ่งกลุ่ม / พบกันหมด
   - Knockout
   - Ranking / เก็บเวลา / เก็บคะแนน
   - Contest / กิจกรรมประกวด
4. แต่ละรายการมีปุ่มเข้าไปจัดการต่อ เช่น:
   - จัดการผล / ตารางกลุ่ม
   - จัดการสาย / บันทึกผล
   - จัดอันดับ / บันทึกผล
   - ให้คะแนน / สรุปผล
5. ถ้ายังไม่มีรายการแข่งขัน จะแสดงกล่อง empty state พร้อมปุ่ม “เริ่มสร้างการแข่งขันแรก”
6. ปรับปุ่ม action บนหน้า Event ให้ความสูงเท่ากัน จัดวางไม่ลอย
