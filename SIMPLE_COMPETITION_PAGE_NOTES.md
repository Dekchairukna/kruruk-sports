# ปรับหน้า Event + สร้างการแข่งขันให้ง่ายแบบหน้าเดียว

ปรับตามคำสั่งล่าสุด:

- หน้า Event เอาปุ่ม Round Robin / Ranking / Knockout ออกจากแถบปุ่มหลัก
- เหลือปุ่มหลักเป็น "สร้างการแข่งขัน"
- ปุ่ม "คลังกีฬา" ยังอยู่แต่เป็นปุ่มรอง ใช้สำหรับแก้คลังกีฬา ไม่ใช่กำหนดระบบแข่ง
- หน้า /events/<event_id>/competitions/quick-new ถูกปรับเป็นแบบง่ายหน้าเดียว
  1. เลือกกีฬา หรือพิมพ์กีฬาใหม่
  2. ทีมในอีเว้นท์ถูกเลือกไว้ทั้งหมดก่อน
  3. เอาทีมที่ไม่ส่ง/ถอนตัวออก
  4. เลือกสูตรแข่ง เช่น พบกันหมด, แบ่ง 4 กลุ่มเอาที่ 1-2, น็อคเอาท์สายติดกัน
  5. กำหนดคะแนน/เซต แล้วกดสร้างการแข่งขัน
- ยังใช้ backend route เดิม ไม่เพิ่มตารางฐานข้อมูลใหม่
- ตรวจ py_compile และ Jinja syntax ผ่าน
