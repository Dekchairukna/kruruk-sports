# Sport Setup User Friendly Update

ปรับหน้า `ตั้งค่ากีฬาในอีเว้นท์` ให้ใช้งานง่ายขึ้นสำหรับผู้ใช้ทั่วไป

## สิ่งที่เพิ่ม
- เพิ่มฟอร์ม "เพิ่มกีฬาเข้าอีเว้นท์แบบง่าย" ที่ทำงานจบในครั้งเดียว
  - เลือกกีฬาเดิม หรือกรอกชนิดกีฬาใหม่
  - เลือก/สร้างหมวดกีฬาได้ทันที
  - เลือกรูปแบบแข่งขันและวิธีบันทึกผลแบบภาษาคนทั่วไป
  - ใส่หลายรุ่นและหลายเพศพร้อมกันได้
  - ระบบสร้าง Sport + SportDivision ให้อัตโนมัติ
- เพิ่มปุ่มลัดกีฬา เช่น ฟุตบอล, ฟุตซอล, วอลเลย์บอล, ตะกร้อ, เปตอง, ชู้ตติ้ง, กรีฑา, ชักเย่อ, ประกวด
- เพิ่มปุ่มลัดรุ่น เช่น ป.1-ป.6, มัธยม, Open
- ซ่อนค่าขั้นสูงของเซต ถ้าไม่ได้เลือกกีฬาที่บันทึกผลแบบรายเซต
- ย้ายแบบฟอร์มเดิมไปไว้ใน "ตั้งค่าขั้นสูง" แบบพับเก็บได้

## Backend ที่เพิ่ม
- เพิ่ม route `sport_quick_add`
- route นี้สร้างหมวดกีฬาใหม่ได้ สร้างชนิดกีฬาใหม่ได้ และสร้างรายการย่อยหลายรายการพร้อมกัน
- ถ้ารายการซ้ำ ระบบจะข้ามและแจ้งจำนวนที่ข้าม
